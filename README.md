# CHUCHOTA

Data-Driven IPA Formant Speech Synthetizer Development Toolkit for WebAudio

WIP. Proof of Concept.

## Use cases

* Formant Speech Synthetizer Development
  * All the required data visible in one UI
  * Finetune, test and copy formants for IPA phonemes

* Create voice personas
  * Tweak base variables for creating presets
  * Override base phonemes for additional variance
  * Copy & Paste new personas for development
